Marcadores de Fuentes

B: Se utiliza para poner en negritas al texto

BIG: Se utiliza para convertir a una fuente más grande

I: Se utiliza para convertir el texto a cursiva

S: Se utiliza para poner una raya en el texto

SMALL: Se utiliza para hacer el texto má s pequeñ 0

STRIKE: Es el mismo resultado que s, pone una raya en el texto

TT: Se utiliza para dar formato de teletipo como se utiliza en los monitores o impresoras antiguas

U: Se utiliza para subrayar una palabra
