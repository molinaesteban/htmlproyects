Frases HTML

EM: se utiliza para dar significado a una parte de un texto

Strong: Se utiliza pra resaltar elementos

Q: se utliza para poner comillas

CODE: Se utliza para poner el texto como si fuera codigo de algun lenguaje de programacion

ABBR: Se utliza para poner Abreviaciones

ACRONYMO : Se utiliza para poner arcónimos

DFN: Se utliza para mostrar Definiciones

KBD: significa texto que debe ser introducido por el usuario

SAMP: Se utiliza para mostrar un ejemplo de una Salida de un programa

VAR: Se utiliza para mostrar que el texto es una Variable
